# 需求文档 (Requirements)
**项目名称**: 仓库管理系统  
**协议版本**: Alkaid-SOP 3.0  
**创建日期**: 2026-01-30  
**状态**: 待确认

---

## 🎯 业务目标
构建一个**未来科技感**的仓库管理系统，支持货物入库、分类管理、查询导出、报表生成、预警和人员管理功能。

---

## 🛠️ 技术栈定义

### 后端
- **框架**: Django (最新稳定版)
- **语言**: Python 3.11+
- **数据库**: SQLite3
- **依赖源**: 清华镜像源 (https://pypi.tuna.tsinghua.edu.cn/simple)

### 前端
- **核心**: HTML5 + CSS3 + JavaScript (原生)
- **架构**: 模块化设计（CSS/JS 完全分离）
- **设计风格**: 未来科技感，主色调蓝色

### 部署
- **容器**: Docker + Docker Compose
- **暴露端口**: 8000 (Django 服务)
- **访问方式**: http://localhost:8000

## 🎨 设计规范

### 视觉风格
- **主题**: 未来科技感
- **主色调**: 蓝色系（#0066FF, #00A3FF, #003D99）

---

## 📦 前端模块化架构（强制规范）

### 必须包含的独立模块
所有页面必须复用以下统一模块：

1. **加载模块** (`loading.js` + `loading.css`)
   - 全局加载动画
   - 支持显示/隐藏切换

2. **提示框模块** (`toast.js` + `toast.css`)
   - 成功/错误/警告/信息提示
   - 自动消失（可配置时长）

3. **表单模块** (`form.js` + `form.css`)
   - 统一表单样式
   - 输入验证（前端）
   - 错误提示显示

4. **按钮模块** (`button.css`)
   - 主要按钮（primary）
   - 次要按钮（secondary）
   - 危险按钮（danger）
   - 禁用状态

5. **菜单栏模块** (`sidebar.js` + `sidebar.css`)
   - 左侧菜单（支持二级菜单）
   - 折叠/展开动画
   - 当前页面高亮

### 严格约束
- ❌ **禁止**：主页 HTML 文件中内嵌 `<style>` 或 `<script>`
- ✅ **要求**：所有样式/脚本通过 `<link>` 和 `<script src>` 引入
- ✅ **要求**：CSS/JS 文件存放在 `static/` 目录

## 🔒 安全与认证

### 认证机制
- Django Session 认证（不使用 DRF Token）
- 所有页面（除登录页）需要 `@login_required` 装饰器
- 前端通过 AJAX 检查登录状态

### 数据安全
- 密码使用 Django 自带的 `make_password` 加密
- 所有 POST 请求必须包含 CSRF Token
- SQL 注入防护（使用 Django ORM）

## 📐 数据库设计（初步）

### 核心表结构
1. **用户表** (Django 自带 `auth_user`)
2. **单位表** (`Unit`): id, name, created_at
3. **品类表** (`Category`): id, name, created_at
4. **品种表** (`Variety`): id, name, category_id, created_at
5. **货物表** (`Goods`): id, name, variety_id, unit_id, quantity, inbound_date, operator, status, created_at
6. **审批记录表** (`Approval`): id, goods_id, approver, status, remark, created_at
7. **人员表** (`Staff`): id, name, role (考勤/出库), created_at

## 🚀 Docker 容器标准

### 交付要求
- **启动命令**: `docker compose up`
- **访问地址**: http://localhost:8000
- **数据持久化**: SQLite 文件通过 Volume 挂载
- **依赖安装**: 使用清华镜像源（pip config）

### Dockerfile 要点
```dockerfile
FROM python:3.11-slim
# 配置清华镜像源
RUN pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
# 安装依赖
COPY requirements.txt .
RUN pip install -r requirements.txt
# 复制项目
COPY . /app
WORKDIR /app
# 初始化数据库
RUN python manage.py migrate
RUN python manage.py createsuperuser --noinput
# 启动服务
CMD ["python", "manage.py", "runserver", "0.0.0.0:8000"]
```
